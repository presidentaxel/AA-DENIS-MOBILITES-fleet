# Configuration des polices Nunito Sans

## Instructions

Pour que la police Nunito Sans fonctionne, vous devez placer les fichiers de polices dans le dossier `frontend/public/fonts/`.

### Fichiers requis :

1. `NunitoSans-VariableFont_YTLC,opsz,wdth,wght.ttf` (police normale)
2. `NunitoSans-Italic-VariableFont_YTLC,opsz,wdth,wght.ttf` (police italique)

### Emplacement :

```
frontend/public/fonts/
├── NunitoSans-VariableFont_YTLC,opsz,wdth,wght.ttf
└── NunitoSans-Italic-VariableFont_YTLC,opsz,wdth,wght.ttf
```

Les fichiers sont déjà référencés dans `frontend/src/styles/global.css` avec les `@font-face` appropriés.

Une fois les fichiers en place, la police sera automatiquement chargée et utilisée sur tout le site.
